📊 Netflix Movies Data Analysis
🔹 Project Overview
This project focuses on analyzing content available on Netflix to uncover trends in movies and TV shows based on genre, release year, country, ratings, and duration.
The goal is to extract meaningful insights that can help understand audience preferences and content distribution.

🔹 Tools & Technologies
Python (Pandas, NumPy)
Matplotlib & Seaborn (Visualization)
Jupyter Notebook
🔹 Conclusion

The analysis reveals that Netflix has significantly expanded its content library in recent years, with a strong focus on diverse genres and international content. This reflects changing audience preferences and global expansion strategies.
